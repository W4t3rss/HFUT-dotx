#### 简介
合肥工业大学本科生毕业论文模板

#### 包含文件

```
HFUT-dotx
├─ GB-T-7714—2015（顺序编码，双语，姓名不大写，无URL、DOI）.csl
├─ HFUT-dotx 说明.pdf
├─ HFUT.dotx
├─ README.md
├─ 原始官网模板_合肥工业大学毕业论文模板.docx
└─ 合肥工业大学博士论文参考文献引用样式.csl
```
